# Review PRD Alignment

Use this skill before or after implementing a feature.

## Check for
- Does implementation match `/docs/killMyIdea_prd.pdf`?
- Are any required fields, states, or flows missing?
- Are any assumptions added without product approval?
- Are edge cases ignored?
- Is the implementation too shallow for a production SaaS app?

## Output format
1. Alignment status
2. Missing requirements
3. Risky assumptions
4. Suggested corrections
